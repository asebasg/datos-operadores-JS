Ejercicio 4: Procesador de texto

Escribe un script que pida al usuario un texto con prompt. Concatena el texto con un saludo constante ("¡Tu texto es: "). Verifica si el texto tiene más de 10 caracteres. Usa const para el saludo, let para el texto y la comparación, y var para la concatenación y una variable con valor null. Aplica un operador lógico ! para negar el resultado de la comparación. Muestra en la consola el texto completo, si es largo, la negación y el valor nulo.
